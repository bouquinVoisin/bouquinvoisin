how to run if you want use sass?
____________

1. install Sass -  http://sass-lang.com/install
2. open comand line / cmd 
3. go to your project directory, enter:

// sass --watch assets/theme/scss/theme.scss:assets/theme/css/theme.css
// sass --watch assets/bootstrap/scss/bootstrap.scss:assets/bootstrap/css/bootstrap.css  

output files you will find in css directory.
3. Enjoy!

This template is using helper classes for margin and padding.
For example:
.mb-5 {margin-bottom: 5px; }

mt-5 - margin top 5px
mr-5 - margin right 5px
mb-5 - margin bottom 5px
ml-5 - margin left 5px


pt-5 - padding top 5px
pr-5 - padding right 5px
pb-5 - padding bottom 5px
pl-5 - padding left 5px

you can use this space amounts: 0, 5, 10, 15, 30, 45, 60
(you can also edit helper mixin in helepers.scss file)
